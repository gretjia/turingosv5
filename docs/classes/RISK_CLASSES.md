# Risk Classes

- Class 0: docs, plans, charters, non-authoritative handoff.
- Class 1: additive isolated helpers, harness docs, schemas, templates, views.
- Class 2: production wire-up, CI, dashboards, replay verifiers, benchmark
  harness code.
- Class 3: auth, money, CAS integrity, capabilities, production evidence.
- Class 4: constitution, flowcharts, sequencer admission, typed tx schema,
  canonical signing payload, trust root, kernel authority.

Class 4 requires exact per-atom human ratification and is never self-selected.
