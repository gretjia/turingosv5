# Class 4 Ratification

Class 4 includes constitution, flowcharts, genesis/trust root, ChainTape refs,
typed transaction schema, sequencer admission, canonical signing payload, and
kernel-level authority.

Class 4 is never self-selected.

Required authorization is exact per-atom human ratification. One-word messages
such as `fix`, `go`, `ok`, `continue`, `can`, `继续`, or `可以` are not Class 4
authorization.
